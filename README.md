Project Summary

Velvet Horizon is a front end business website developed for a travel agency as part of the course AICT Applications of Information and Communication Technology. The website presents travel packages, services, and contact information in a clean and user friendly layout. The project demonstrates practical use of HTML and CSS along with live deployment using GitHub Pages.

This repository includes the website source code and supporting academic documentation including a Word report, Excel data file, and presentation slides. All components are aligned and reference the same data and design decisions.
